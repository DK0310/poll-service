Everything you need to run a live poll

All the features built into PollBuilder — pick a question type, share, and engage.

▦

Four question types

Multiple choice, Yes / No, Rating 1–5, and Open text — with 2–6 options where it counts.

⚡

Live results

Bar charts update the instant anyone votes, streamed over WebSockets (SignalR).

🔗

Share a short link

Every poll gets a 5-character code like /poll/7fGh2. Vote once per browser, no login.

💬

Anonymous Q&A

The audience asks questions and upvotes; the host can pin the best — all live.

📊

Creator analytics

Votes over time, peak voting minute, and the top option for every poll you own.

⏲️

Auto-close on expiry

Set a poll to close in an hour, a day, or a week — final results lock automatically.

How it works

From idea to live results in three steps.

1

Create

Sign in, write your question, pick a type, add options and an optional expiry.

2

Share

Send the short /poll/code link. Anyone can vote — no account required.

3

Watch live

Results and Q&A update in real time. Dig into analytics when you're done.
