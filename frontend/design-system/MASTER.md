## Color tokens

Semantic tokens — never hardcode raw hex in components. Light, rose-tinted theme (no dark mode v1).

| Role | Hex | CSS var | Notes / contrast |
|---|---|---|---|
| Rose (primary) | `#E11D48` | `--rose` | dominant accent: CTAs, selected, leading bar, live dot |
| Rose soft | `#FB7185` | `--rose-2` | gradient partner, hovers |
| Rose deep | `#be123c` | `--rose-deep` | **small red text** — ~5:1 on bg ✓ |
| Blue (accent) | `#2563EB` | `--blue` | engagement secondary: links, analytics line, focus ring |
| Violet | `#A855F7` | `--violet` | mesh-gradient third stop only (decorative) |
| Background | `#FFF1F2` | `--bg` | rose-tinted base; mesh gradient sits behind |
| Surface (glass) | `rgba(255,255,255,.72)` | `--surface` | card fill over the mesh (backdrop-blur) |
| Surface border | `rgba(255,255,255,.65)` | `--surface-bd` | glass edge highlight |
| Ink (body) | `#2e1620` | `--ink` | body text — ~12:1 on bg ✓ AAA (neutral plum, not pure rose) |
| Ink soft | `#5b3a46` | `--ink-soft` | secondary text — ~6:1 ✓ |
| Muted | `#7c5260` | `--muted` | captions/meta — ~4.6:1 ✓ AA |
| Success | `#047857` | `--success` | confirmations (paired with ✓ icon) |
| Danger | `#DC2626` | `--danger` | destructive (paired with text/icon) |

> **Contrast rule:** body/long text uses `--ink`/`--ink-soft` on a **solid glass surface**, never directly on the
> gradient. Reserve `--rose`/gradients for large headings, fills, and CTAs. Status is never color-only (pills carry a label).

## Typography

- **Display:** `"Space Grotesk"` — wordmark, headlines, questions, big numerals. 500/600/700. (skill's pick)
- **Body:** `"DM Sans"` — paragraphs, options, labels. 400/500/700.
- **Mono:** `"DM Mono"` — poll codes, tallies, percentages, dates, the LIVE badge. 400/500. (pairs with DM Sans; tabular)
- Self-host via `@fontsource/space-grotesk`, `@fontsource/dm-sans`, `@fontsource/dm-mono` (no CDN → Docker/offline-safe). `font-display: swap`.
- Base 16px / line-height **1.55** body; headings 1.05–1.15, `letter-spacing:-0.02em`. **Tabular figures** on all counts/percentages.

```
--font-display: "Space Grotesk", system-ui, sans-serif;
--font-body:    "DM Sans", system-ui, sans-serif;
--font-mono:    "DM Mono", ui-monospace, monospace;
```

**Type scale (px):** 12 · 14 · 16 · 18 · 22 · 28 · 36 · 48 · 64. Weight: headings 600–700, body 400, labels 500.

## Spacing (4/8 rhythm)

`--space-xs 4` · `--space-sm 8` · `--space-md 16` · `--space-lg 24` · `--space-xl 32` · `--space-2xl 48` · `--space-3xl 64`.

## Surfaces, radii & elevation

Aurora = **glassy, rounded, soft-colored shadows** over a living gradient.

- Radii: `--radius-sm 12px` · `--radius 16px` · `--radius-lg 22px` · pills 999px.
- Glass card: `background:var(--surface); backdrop-filter:blur(16px); border:1px solid var(--surface-bd)`.
- Shadow (soft, tinted, not black): `--shadow: 0 24px 60px -28px rgba(136,19,55,.45)`; CTA glow `0 14px 30px -12px rgba(225,29,72,.6)`. One consistent scale.
- **Mesh gradient background** (fixed, behind content): layered radial-gradients in rose/blue/violet, blurred ~20px.

## Motion tokens

- `--ease-enter: cubic-bezier(.2,.7,.3,1)` · `--ease-exit: cubic-bezier(.4,0,1,.6)`.
- Durations: micro `--dur-1 180ms`, state `--dur-2 240ms`; never > 400ms. Exit ≈ 70% of enter.
- Animate **transform/opacity only**. Stagger list reveals 30–50ms. Max 1–2 animated elements per view.
- Signatures: slow **mesh drift** (~12–14s, `transform` only), bar fills ease in, live dot pulse, gradient-text headings, subtle card hover lift.
- **`@media (prefers-reduced-motion: reduce)`** → freeze mesh drift, disable pulse/lift, show final state instantly. *(skill: avoid excessive animation.)*

## Charts & data (Analytics R5 / Results R4)

From the skill's chart DB (trend-over-time + accessibility):

- **Votes over time → line chart.** Blue (`--blue`) stroke, ~20% area fill, hairline low-contrast grid, mono axis labels with time granularity.
- **Distinguish series by line-style (solid/dashed), not color alone.** Data text labels ≥ 4.5:1; data line vs bg ≥ 3:1.
- Provide a **toggleable data-table fallback** (timestamps + values) + `aria-label` summary for screen readers.
- **Empty state** ("No votes yet" + guidance), never a blank axis frame. **Loading** = skeleton, not an empty frame.
- Results bars: leading option = rose gradient, others = blue/violet gradients; exact count + % (tabular). Total votes as a large **KPI**.
- Freeze chart entrance animation under `prefers-reduced-motion`.

## Anti-patterns (do NOT use)
- ❌ Emoji as structural icons → use **Lucide SVG** icons (one consistent stroke width).
- ❌ Body/long text directly on the gradient (contrast) → always on a glass surface.
- ❌ Dark-by-default (skill: Aurora is a light theme); excessive/constant animation.
- ❌ Placeholder-only labels; errors only at top of form.
- ❌ Color-only status; invisible focus rings; instant 0ms state changes.
- ❌ Animating width/height/top/left (use transform/opacity).
- ❌ Space Grotesk **everywhere at tiny sizes** — keep it for display; DM Sans for body.

## Pre-delivery checklist (every phase)
- [ ] Text contrast ≥ 4.5:1 (`--ink`/`--ink-soft`/`--muted`/`--rose-deep` on glass; never small text on gradient)
- [ ] Visible `:focus-visible` rings (blue); full keyboard nav; tab order = visual order
- [ ] Touch targets ≥ 44×44px, ≥ 8px gaps; `cursor:pointer` on clickables
- [ ] Lucide SVG icons only (no emoji); one consistent stroke width
- [ ] Transitions 150–300ms, transform/opacity only; `prefers-reduced-motion` honored (mesh freezes)
- [ ] Form labels visible; errors below field with `role="alert"`; submit shows loading→result
- [ ] Charts: data-table fallback, line-style not color-only, empty/loading states
- [ ] Responsive at 375 / 768 / 1024 / 1440; no horizontal scroll; container max-width
- [ ] Tabular figures on all counts/percentages
